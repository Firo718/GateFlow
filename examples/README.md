# GateFlow Examples

These examples are the minimum runnable references for the core capability areas.

- Basic project: `examples/simple_project.py`
- TCP build flow: `examples/build_zed_led_tcp.py`
- Reports and lint checks: `examples/report_checks_example.py`
- Simulation debug flow: `examples/simulation_debug_example.py`
- IP query + minimal simulation closure: `examples/ip_query_simulation_example.py`
- Block design templates: `examples/zynq_template_example.py`
- Hardware debug runtime: `examples/hardware_debug_runtime_example.py`

Each file is intentionally small and focuses on one workflow that should be easy to adapt.
